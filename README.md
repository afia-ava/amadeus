# Amadeus-robot

Amadeus is an observational robot that can learn from the environment through a combination of video, audio, and other sensors. Its purpose is to learn continuously from lived experiences and use that learning to train a deeply personalized AI assistant.

**Overview**
- a passive robot
- oberseves the surrounding
- learns over time and gets smarter

**Goals**
- train an AI assistant 
- enable human-like learning 
- continuously imrpove the memory 

**System Architecture**

Hardware:
- camera, microphones, IMU, GPS sensors
- ai SoC for compute

Software:
- data ingestion and processing
- vector database for knowledge and memory
